What is CSS?

CSS (Cascading Style Sheets) is a language used to define the visual appearance of a web page written in HTML or XML. It controls the layout, color schemes, fonts, spacing, and other visual aspects of a web page. Without CSS, web pages would be plain and difficult to navigate, giving a poor user experience.

Inline CSS

Inline CSS is used to apply styles directly to a specific HTML element using the style attribute. This style will only affect that particular element.

Embedded CSS

Embedded CSS refers to styles that are written inside a <style> tag within the <head> section of an HTML document. It’s a great option when you want to style elements within a single document.

            <style>
                .example-class {
                    color: #ff1493;
                    font-weight: bold;
                }
            </style>

This text is styled using Embedded CSS!

External CSS

External CSS is the most efficient way to apply styles to multiple pages of a website. You link a separate CSS file using the <link> tag in the <head> section of the HTML file.

            <link rel="stylesheet" href="styles.css">
